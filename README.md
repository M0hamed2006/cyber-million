# Pulse Party — الحقيقة أو التحدي

لعبة بارتي محلية بتصميم Glassmorphism وواجهة احترافية.

## المميزات
- إضافة وحذف لاعبين
- أوضاع لعب متعددة
- بنك أسئلة آمن وخفيف
- حفظ تلقائي في المتصفح
- مؤثرات صوتية واحتفالية
- واجهة متحركة ومناسبة للعرض على GitHub

## الملفات
- `index.html`
- `style.css`
- `script.js`

## التشغيل
افتح `index.html` في المتصفح.
