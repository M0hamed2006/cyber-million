## 🚀 البدء السريع

### المتطلبات
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- اتصال إنترنت (اختياري - التطبيق يعمل بدون إنترنت بعد التحميل الأول)

### التثبيت المحلي

1. **استنسخ المستودع:**
```bash
git clone https://github.com/your-username/pulse-party-ultra.git
cd pulse-party-ultra
```

2. **ابدأ خادم محلي:**
```bash
# Python 3
python -m http.server 8000

# أو Node.js
npx http-server

# أو Live Server في VS Code
```

3. **افتح المتصفح:**
```
http://localhost:8000
```

### النشر على GitHub Pages

1. **إنشء مستودع GitHub**
2. **ادفع الملفات:**
```bash
git push origin main
```

3. **فعّل Pages:**
   - اذهب لإعدادات المستودع
   - اختر GitHub Pages
   - اختر الفرع main
   - احفظ التغييرات

4. **الرابط:**
```
https://username.github.io/pulse-party-ultra
```

## 📖 تعليمات الاستخدام

### خطوة 1️⃣: إضافة اللاعبين
```
1. اكتب اسم اللاعب في الحقل
2. اضغط ➕ أو Enter
3. كرر للاعبين الآخرين
4. يجب على الأقل لاعبين!
```

### خطوة 2️⃣: اختر الإعدادات
```
- نوع السؤال (عشوائي/حقيقة/تحدي/إلخ)
- مستوى الصعوبة (سهل/متوسط/صعب/قاسي/أسطوري)
- عدد الأسئلة (5-100)
- سرعة اللعبة (بطيء/عادي/سريع/مجنون)
```

### خطوة 3️⃣: ابدأ اللعبة
```
اضغط 🎲 ابدأ اللعبة الآن!
```

### خطوة 4️⃣: اللعب
```
1. يسأل السؤال للاعب الأول
2. اللاعب يجيب
3. اضغط ✅ تم الإجابة
4. ننتقل للاعب التالي
5. كرر حتى ينتهي العدد المحدد
```

### التحكم أثناء اللعبة
- **✅ تم الإجابة** - الانتقال للسؤال التالي
- **⏭️ تخطي** - تخطي هذا السؤال
- **🔄 إعادة** - تكرار نفس السؤال
- **✏️ سؤال مخصص** - إضافة سؤالك الخاص
- **⏸️ إيقاف مؤقت** - إيقاف اللعبة مؤقتاً
- **🛑 إنهاء** - إنهاء اللعبة

## 🐛 حل المشاكل

### المشكلة: لا تعمل الموسيقى
**الحل:** تحقق من أن الصوت مفعل، والتزم بسياسة autoplay الحديثة (اضغط الزر أولاً)

### المشكلة: التسجيل الصوتي لا يعمل
**الحل:** 
1. تأكد من منح إذن الميكروفون
2. استخدم HTTPS أو localhost
3. حاول متصفح آخر

### المشكلة: البيانات لم تُحفظ
**الحل:** 
1. تحقق من أن localStorage مفعل
2. تأكد من عدم وجود وضع التصفح الخاص
3. مسح الذاكرة المؤقتة قد يساعد

### المشكلة: الصفحة بطيئة
**الحل:**
1. أغلق التطبيقات الأخرى
2. مسح ذاكرة التخزين المؤقت
3. حدّث المتصفح
