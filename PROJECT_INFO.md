```javascript
STORAGE_KEYS {
  PLAYERS: 'ppa_players'      // قائمة اللاعبين

  STATS: 'ppa_stats'          // الإحصائيات
  SETTINGS: 'ppa_settings'    // الإعدادات
  SESSIONS: 'ppa_sessions'    // الجلسات السابقة
}
```

### 🎵 نظام الصوت
```javascript
- مؤثرات صوتية عند الإجابة
- نغمات مختلفة لكل حدث
- موسيقى خلفية منخفضة الصوت (25% افتراضي)
- تحكم كامل بمستوى الصوت
- 4 أنماط موسيقية مختلفة
```

### 🎤 تسجيل صوتي
```javascript
- تسجيل حتى 10 ثوانٍ
- إيقاف تلقائي
- تشغيل فوري
- حفظ في الدردشة
```

### 📸 مشاركة الصور
```javascript
- تحميل صور محلية

- حفظ في الدردشة
```

### 🎉 تأثيرات الكونفيتي

